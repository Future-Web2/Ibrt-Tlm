3D Book Animation Component

Интерактивный компонент с 3D анимацией прокрутки

О компоненте

Это полностью готовый компонент 3D книги с анимацией при прокрутке. Книга начинает закрытой и постепенно открывается при прокрутке страницы вниз.

Компонент полностью адаптивен и работает на всех устройствах. Используйте CSS переменную --scroll-progress для управления анимацией.

Как использовать

Просто скопируйте файлы book-3d.html, book-3d.css и book-3d.js в вашу папку и включите их в ваш основной проект.

Все CSS стили полностью кастомизируемы. Отредактируйте цвета, размеры и текст под ваши нужды.
