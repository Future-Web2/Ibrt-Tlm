# 📦 Полный пакет для интеграции

Компонент готов к использованию! Вот что вы получили:

## 📂 Содержимое папки `book-component`

```
book-component/
├── book-3d.html         ← HTML структура (копируй нужные части)
├── book-3d.css          ← Все стили компонента
├── book-3d.js           ← Логика анимации
├── example.html         ← Пример использования
├── package.json         ← Информация о компоненте
└── README.md            ← Подробная документация
```

---

## 🎯 Быстрый старт (3 шага)

### 1️⃣ Добавьте CSS в <head>
```html
<head>
    <link rel="stylesheet" href="path/to/book-3d.css">
</head>
```

### 2️⃣ Добавьте HTML в <body>
Скопируйте всю `<section class="book-scroll-track">` из `book-3d.html` 
в место, где должна быть книга.

### 3️⃣ Добавьте JavaScript перед </body>
```html
<script src="path/to/book-3d.js"></script>
</body>
```

---

## 🎨 Персонализация

### Измените логотип
В `book-3d.html` найдите:
```html
<div class="book-cover left-back-cover" 
     style="background-image: url('YOUR_LOGO_HERE')">
</div>
```

### Измените текст
Отредактируйте содержимое `.static-page` и `.flipping-page`.

### Измените цвета
В `book-3d.css` изменяйте:
- `#4a2810` - цвет обложки
- `#c59b27` - золотой цвет бордюра
- `#fbf7ee` - цвет страниц
